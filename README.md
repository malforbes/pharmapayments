# PharmaPayaments

This repository analyses pharmaceutical company payments to Australian clinicians from 2015–2024.

## Overview

The project focuses on cleaning, and analysing publicly available disclosure data released by pharmaceutical companies in Australia. 
The goal is to support transparent, reproducible analysis of industry payments to healthcare professionals.

## Repository structure

PharmaPayments/
├── raw-data/
├── scripts/
└── manuscript/

- **raw-data/**  
  Original CSV files - read-only. 

- **scripts/**  
  R workflows plus a Python web scraper used for accuracy checks. 

- **manuscript/**  
  Pre-print with supplementary information.
