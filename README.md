# PharmaPayments

This repository analyses pharmaceutical company payments to Australian clinicians from 2015–2024.

## Overview

The project analyses publicly available disclosure data released by pharmaceutical companies in Australia.  
The goal is to support transparent, reproducible analysis of industry payments to healthcare professionals.
